# telecom-plan-revenue-analysis

## Technical Highlights

· Statistical Profiling: Aggregated monthly usage data (minutes, SMS, gigabytes) per user to calculate true average revenue per user (ARPU).

· Inferential Statistics: Performed Two-Sample T-Tests (using scipy.stats) to determine if revenue differences between plans were statistically significant.

· Outlier Management: Utilized the Interquartile Range (IQR) method to identify and handle usage anomalies that could skew the business conclusion.
